# Images

Shared image assets referenced from the top-level `README.md` and from
sibling folders' READMEs.

## What goes here

- Hero photos of the MVP build (Jetson + LyraT wired together).
- Rendered architecture diagrams (the 8 workflow diagrams sourced from
  the prompts in
  [genie-claw / doc / workflow / prompt.md](https://github.com/GeniePod/genie-claw/blob/main/doc/workflow/prompt.md)).
- Logos / wordmarks (also mirrored in `../product-design/identity/`
  to keep that folder self-contained).

## Conventions

- **Filenames:** kebab-case, lowercase, descriptive. `hero-mvp.jpg`
  not `IMG_0241.HEIC`. Suffix with `-srgb.png` / `-cmyk.tif` if the
  same image is checked in for both screen and print.
- **Sizes:** prefer ≤ 2 MB per image. Use PNG for diagrams and screen
  renders, JPG for photographs (quality 85), SVG for logos and vector
  diagrams.
- **License:** images created for this repo are CERN-OHL-S v2 like
  everything else. Photos that include third-party trademarks (Jetson
  branding, NVIDIA logos) are fair-use illustrative and must be
  captioned accordingly when reused.
